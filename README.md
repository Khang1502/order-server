# ORDER App - Fullstack (Demo)

Hướng dẫn:

1. Backend:
   - cd server
   - npm install
   - npm start
   Server chạy: http://localhost:3000

2. Frontend:
   - cd client
   - npm install
   - npm run dev
   Frontend chạy (Vite): http://localhost:5173

Lưu ý:
- Đây là bản demo đơn giản. Bạn có thể nâng cấp bằng authentication, cart, multi-item orders, deploy, etc.
